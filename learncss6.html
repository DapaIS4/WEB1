Learning CSS

CSS stands for Cascading Style Sheets. It is a stylesheet language used to control the presentation and formatting of web documents, including HTML and XML documents. CSS allows web developers to separate the structure and content of a web page (defined by HTML) from its visual style and layout (defined by CSS).
